What is a flippet block ?

Web friendly, we use Codepen for everything !

For some reasons, every demos, and links to codes will be powered by codepen, it’ll be faster, and it permit us to add more elments to the collections, for example, if you want to add an element that use Saas, Compass, less, or anything else, codepen can handle it, and provide a clean version of the code.

Everyone can contribute

Flippets tends to be the most advanced collection of flat User Interfaces elements, so its pretty normal to allow users to send their own works.

All contributions will be review and selected, only if they correspond to some criters, and if it respect the Flippets edition rules.
